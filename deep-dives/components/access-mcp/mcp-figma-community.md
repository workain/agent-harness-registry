# Figma-Context-MCP (community)

**Registry entry:** `data/components/mcp-figma-community.yaml` · **Category:** access-mcp

## What it is

The popular COMMUNITY Figma-context server, predating/paralleling Figma's own official hosted server — higher star count (15.3k) than several official entries in this batch.

## When to use it

You want self-hostable Figma design-context access rather than depending on Figma's own hosted beta server.

## How to get started

Follow the repo's Figma API token setup.

## Gotchas

- Directly compare against Figma's own official hosted server (see this registry's `mcp-figma-official` entry) before choosing — community vs. official is a real maturity/openness tradeoff here, not just a naming difference.

## How it compares

See `mcp-figma-official` in this same registry for the vendor-hosted alternative — same problem, two very different maturity/openness tradeoffs.
