# Figma MCP (official)

**Registry entry:** `data/components/mcp-figma-official.yaml` · **Category:** access-mcp

## What it is

Figma's own official hosted MCP server (closed-source) plus a local desktop-app variant. Currently free during beta; will become usage-based paid.

## When to use it

You want first-party support and are comfortable with the beta-to-paid pricing transition, over the community alternative's greater maturity/self-hosting.

## How to get started

Connect via `mcp.figma.com` per Figma's own docs, or use the local desktop-app variant.

## Gotchas

- Currently free during beta — budget for usage-based pricing once that period ends.

## How it compares

Directly contrast with the community `Figma-Context-MCP` entry in this registry (15.3k stars, MIT, self-hostable, predates this official server) — same problem, opposite openness/maturity tradeoffs.
