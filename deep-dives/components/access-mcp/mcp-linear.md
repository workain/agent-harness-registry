# Linear MCP

**Registry entry:** `data/components/mcp-linear.yaml` · **Category:** access-mcp

## What it is

Linear's official hosted MCP server, built in partnership with Cloudflare and Anthropic — OAuth 2.1, 25+ tools for issues/projects/teams/comments/workflow states.

## When to use it

Your team tracks work in Linear and you want an agent managing issues/projects directly.

## How to get started

Connect via `mcp.linear.app` per Linear's own docs; OAuth 2.1 handles authentication.

## Gotchas

- No public repo found — same hosted-only, no-self-host pattern as Zapier MCP in this registry; you depend entirely on Linear's own infrastructure.

## How it compares

Same hosted-only, vendor-locked pattern as Zapier MCP and the official Figma MCP server in this registry.
