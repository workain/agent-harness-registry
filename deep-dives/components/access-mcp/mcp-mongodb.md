# MongoDB MCP Server

**Registry entry:** `data/components/mcp-mongodb.yaml` · **Category:** access-mcp

## What it is

Official (`mongodb-js` org) server for interacting with MongoDB databases from an agent.

## When to use it

Your data layer is MongoDB and you want direct agent access to it.

## How to get started

Connect to an existing MongoDB deployment per the repo's setup docs.

## How it compares

Same database-backend-server pattern as Supabase/Redis/Elasticsearch in this registry.
