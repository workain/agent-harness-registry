# Pipedream

**Registry entry:** `data/components/mcp-pipedream.yaml` · **Category:** access-mcp

## What it is

A Composio/Zapier-MCP-adjacent aggregation platform — 2,500+ APIs, 8,000+ prebuilt tools — but crucially SELF-HOSTABLE, unlike Zapier MCP's hosted-only model.

## When to use it

You want Zapier-MCP-scale app connectivity but need to self-host rather than depend on a vendor's hosted service.

## How to get started

Self-host per the repo's docs, or use Pipedream's own hosted offering if you don't want to run infrastructure.

## Gotchas

- License is NOASSERTION on GitHub — verify actual terms, especially if self-hosting for a commercial product.

## How it compares

Directly contrast with Zapier MCP (hosted-only, 9,000+ apps, no self-host option) and Composio (1,000+ toolkits, framework-provider packages) — Pipedream sits between them: broader than Composio, self-hostable unlike Zapier.
