# AWS MCP Servers

**Registry entry:** `data/components/mcp-aws.yaml` · **Category:** access-mcp

## What it is

A SUITE of specialized AWS-focused MCP servers in one monorepo (AWS API, Documentation, Knowledge, Data Processing, and more) — not a single server.

## When to use it

You're operating AWS infrastructure and want an agent equipped with AWS-specific tool access and up-to-date documentation lookup.

## How to get started

Pick the specific sub-server(s) you need from the monorepo rather than installing the whole suite.

## Gotchas

- AWS's own docs point to a separate 'Agent Toolkit for AWS' as the recommended path going forward — check current guidance before committing to this repo as your long-term integration point.

## How it compares

The broadest official-vendor MCP offering in this registry by sub-server count.
