# Elasticsearch MCP Server

**Registry entry:** `data/components/mcp-elasticsearch.yaml` · **Category:** access-mcp

## What it is

Elastic's official server for querying and managing Elasticsearch data through natural language.

## When to use it

Your search/log infrastructure runs on Elasticsearch and you want an agent to query it directly.

## How to get started

Connect to an existing Elasticsearch cluster per the repo's setup docs.

## How it compares

One of several official database/search-backend servers in this registry (also see Supabase, MongoDB, Redis).
