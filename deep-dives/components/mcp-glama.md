# Glama (MCP directory + hosting)

**Registry entry:** `data/components/mcp-glama.yaml` · **Category:** access-mcp

## What it is

A registry+hosting combo: indexes 51,230+ servers per its own claim, and offers one-click hosting (isolated VM per server) — distinct from Smithery (registry+CLI, no hosting) in this registry.

## When to use it

You want to both DISCOVER an MCP server and have somewhere to run it without managing your own hosting, in one product.

## How to get started

Browse glama.ai directly; no GitHub repo to install from for the core service.

## Gotchas

- No public repo found for the core registry/hosting service — server counts and hosting claims are the vendor's own, not independently verified here.

## How it compares

Registry+hosting, vs. Smithery's registry+CLI (no hosting) and the Official MCP Registry's registry-only (no hosting, no CLI) in this same category.
