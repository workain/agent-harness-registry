# Official MCP Registry

**Registry entry:** `data/components/mcp-official-registry.yaml` · **Category:** access-mcp

## What it is

"A community driven registry service for MCP servers," under the `modelcontextprotocol` org itself — the same steering group as this registry's `mcp-servers`/`mcp-client-sdk` entries. The canonical registry other directories (Glama, PulseMCP) build on top of or aggregate from.

## When to use it

You want the most authoritative available server registry, maintained by the protocol's own steering group, rather than a third-party aggregator.

## How to get started

Query the registry API/browse per the repo's docs.

## Gotchas

- License is NOASSERTION on GitHub — verify before redistributing registry data.

## How it compares

More authoritative than third-party directories (Glama, PulseMCP) in this same registry, since it's maintained by MCP's own steering group.
