# Postgres MCP Pro

**Registry entry:** `data/components/mcp-postgres-pro.yaml` · **Category:** access-mcp

## What it is

A Postgres server spanning the whole dev lifecycle — configurable read/write access plus performance analysis, not just querying.

## When to use it

You need more than basic query access — genuine performance-analysis tooling alongside standard read/write operations.

## How to get started

Connect to your Postgres instance per the repo's setup docs.
