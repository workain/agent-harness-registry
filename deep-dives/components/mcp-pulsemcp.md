# PulseMCP

**Registry entry:** `data/components/mcp-pulsemcp.yaml` · **Category:** access-mcp

## What it is

A proprietary web directory claiming 20,000+ servers updated daily, aggregating from the official registry plus scraping/manual curation.

## When to use it

You want the most frequently-updated aggregator view, if you're comfortable with a claim that isn't independently verified.

## How to get started

Browse pulsemcp.com directly; no GitHub repo to install from.

## Gotchas

- 'Updated daily' and the 20,000+ count are the vendor's own claims — not independently verified in this research pass.

## How it compares

Positioned as the most actively-updated aggregator by claimed update frequency versus Glama/the Official Registry — but this specific claim wasn't independently checked.
