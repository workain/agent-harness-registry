# slack-mcp-server (community)

**Registry entry:** `data/components/mcp-slack-community.yaml` · **Category:** access-mcp

## What it is

A community Slack server billed as the 'most powerful' available — no permission-request friction, supports GovSlack, DMs, and group DMs.

## When to use it

You want fuller Slack API access than a narrowly-scoped official integration provides, and don't need first-party vendor support.

## How to get started

Follow the repo's Slack app/token setup instructions.

## Gotchas

- Community-maintained, not an official Slack product — evaluate security/maintenance posture independently before granting it broad workspace access.

## How it compares

No official first-party Slack MCP server was found in this research pass — this community entry is the most-adopted Slack option currently cataloged.
