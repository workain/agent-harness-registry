# vercel-labs/skills

**Registry entry:** `data/components/skill-vercel-skills-cli.yaml` · **Category:** skills-tools

## What it is

Vercel's own open agent-skills INSTALLER tool (`npx skills`) — not a skill itself, the CLI other Vercel skill entries in this registry install through. Includes a `find-skills` meta-skill for discovery.

## When to use it

You want Vercel's own installer/discovery tool for the skills ecosystem.

## How to get started

`npx skills` to browse and install; `find-skills` for discovery.
