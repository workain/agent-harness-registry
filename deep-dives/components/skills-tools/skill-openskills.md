# numman-ali/openskills

**Registry entry:** `data/components/skill-openskills.yaml` · **Category:** skills-tools

## What it is

A universal skills loader bringing Anthropic's SKILL.md format to ANY agent that can read AGENTS.md (Cursor, Windsurf, Aider) — not just Claude Code.

## When to use it

You want to try the Agent Skills format without committing to Claude Code specifically.

## How to get started

`npx openskills install anthropics/skills` — the single most useful onboarding command for trying skills on a non-Claude-Code engine.

## Gotchas

- License is NOASSERTION — verify before redistributing.

## How it compares

The cross-engine counterpart to Vercel's own `npx skills` installer — pick this specifically if you're NOT on Claude Code.
