# obra/superpowers

**Registry entry:** `data/components/skill-superpowers.yaml` · **Category:** skills-tools

## What it is

An agentic skills framework and software-development methodology — not a single skill but a composable framework chaining brainstorming -> git-worktree setup -> implementation planning -> subagent-driven execution -> TDD -> code review. The largest community-built skill library found in this research by star count.

## When to use it

You want a full opinionated development methodology encoded as chainable skills, not just one narrow capability.

## How to get started

Adopt incrementally — start with just the TDD or code-review sub-skill rather than the whole methodology at once, or it will dominate your project's CLAUDE.md.

## Gotchas

- Heaviest-weight entry in this registry's skills-tools category — installing the whole thing at once is a lot to absorb; incremental adoption is the practical path.

## How it compares

Conceptually overlaps with `ai-coding-project-boilerplate` in this registry's bundles (also a full dev methodology), but ships as the more portable Agent Skills format rather than a fixed subagent roster.
