# camel-ai/camel

**Registry entry:** `data/components/subagent-camel.yaml` · **Category:** subagents

## What it is

A two-agent role-playing primitive (`role_playing.py`) that spawned a broader "Finding the Scaling Law of Agents" framework and the `camel-ai/owl` (Optimized Workforce Learning) follow-on project.

## When to use it

Researching or building on the role-playing primitive itself, or exploring CAMEL's scaling-law framework.

## How to get started

Follow the repo's docs for `role_playing.py`.

## Gotchas

- Mixed license — code is Apache-2.0 (permissive) but any bundled DATASETS are CC-BY-NC-4.0 (non-commercial only). Check which you're actually using before a commercial deployment.

## How it compares

A research-oriented two-agent primitive, narrower in immediate practical scope than MetaGPT/ChatDev's full role pipelines.
