# VoltAgent/awesome-claude-code-subagents

**Registry entry:** `data/components/subagent-voltagent-collection.yaml` · **Category:** subagents

## What it is

100+ subagents organized into 10 categories (core-dev, language-specialists, infra, quality-security, data-ai, dev-experience, specialized-domains, business-product, meta-orchestration, research-analysis) — the best-organized taxonomy of any collection found in this research.

## When to use it

You want a broad, well-categorized subagent starting point and would rather browse by domain than build a roster from scratch.

## How to get started

Copy individual `.md` files into `.claude/agents/` — don't bulk-import all 100+ at once (real discovery/context overhead); check the repo's `tools/subagent-catalog` for a machine-readable index first.

## Gotchas

- Bulk-installing everything defeats the purpose — Claude has to consider every installed subagent's description when routing, so more isn't free.

## How it compares

The best-organized taxonomy in this registry's subagent collections — use as a category reference even if you don't adopt the files wholesale.
