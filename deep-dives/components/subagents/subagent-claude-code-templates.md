# davila7/claude-code-templates

**Registry entry:** `data/components/subagent-claude-code-templates.yaml` · **Category:** subagents

## What it is

A CLI + dashboard (aitmpl.com) for browsing/installing 100+ agents, commands, MCPs, and hooks as one unified marketplace UI — the most product-like discovery experience of any collection researched.

## When to use it

You want a visual/interactive way to browse and install subagents rather than reading raw markdown files on GitHub.

## How to get started

`npx claude-code-templates` for interactive install. The companion `davila7/claude_subagents` repo is a hands-on playground specifically for testing context-isolation behavior (clean/forked/nested up to 5 levels).

## How it compares

The most polished discovery UX of any subagent collection in this registry — worth starting here if you find raw-markdown browsing tedious.
