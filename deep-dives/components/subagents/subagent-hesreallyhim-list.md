# hesreallyhim/a-list-of-claude-code-agents

**Registry entry:** `data/components/subagent-hesreallyhim-list.yaml` · **Category:** subagents

## What it is

A curated LIST of community-submitted subagents (a discovery index, not authored agents itself).

## When to use it

As a discovery feed to find subagents other people have published, rather than a source of agents itself.

## How to get started

Browse the list and follow links to the actual subagent repos it indexes.

## How it compares

A list, like `awesome-claude-code` (below) but scoped specifically to subagents rather than the whole Claude Code ecosystem.
