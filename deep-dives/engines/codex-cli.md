# OpenAI Codex CLI

**Registry entry:** `data/engines/codex-cli.yaml`

## What it is

OpenAI's lightweight terminal coding agent, rewritten in Rust. Runs locally, drives the model
turn-by-turn against a local repo. Single-binary distribution.
