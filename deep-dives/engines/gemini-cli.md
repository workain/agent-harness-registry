# Gemini CLI

**Registry entry:** `data/engines/gemini-cli.yaml`

## What it is

Google's open-source terminal AI agent bringing Gemini into the command line. Reads hierarchical
GEMINI.md instruction files (global -> project/ancestor -> subdirectory, more specific overriding
general), supports MCP tools, and ships `/init` to bootstrap a starting instruction file.

## Equipment surface

The engine this registry's `gemini-md` component documents in detail (the hierarchical
instruction-loading convention specifically).
