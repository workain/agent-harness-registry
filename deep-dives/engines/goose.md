# Goose

**Registry entry:** `data/engines/goose.yaml`

## What it is

An open-source, general-purpose AI agent from Block (Square) — install, execute, edit, and test
with any LLM. Supports 15+ LLM providers and 70+ MCP extensions. Recently transitioned from
`block/goose` to the Agentic AI Foundation (AAIF) under the Linux Foundation — a real instance of
an agent engine moving to neutral foundation governance.

## License

## Equipment surface

The extension point this registry's `.goosehints` component plugs into (project instructions);
Goose's own MCP integration is one of the deepest in the ecosystem.
